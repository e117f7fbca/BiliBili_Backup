# BiliBili_Backup

一个备份B站UP主内容的脚本，低技术力，仅供参考

备份内容:
> UP主空间  
>> 1.头像（face.jpg）  
>> 2.banner（top_photo.jpg）  
>> 3.关注数（stat.json）  
>> 4.个人介绍（info.json）  

> 动态  
>> 1.动态原数据（dynamic.json）（内容、浏览、点赞、转发、原动态）  
>> 2.动态评论（comment.json）（评论区内容）  
>> 3.动态点赞列表（like.json）(点赞用户ID，点赞时间)  
>> 4.图片（/picture/1.jpg）  
>> 5.转发动态原动态（/origin/dynamic）  
>> 6.小视频动态(svideo.mp4)  

> 视频  
>> 1.视频分P（cidXXXXXXX）（调用annie下载）  
>> 2.视频信息（info.json）  
>> 3.视频统计（stat.json）  
>> 4.视频tag（tag.json）  
>> 5.分P信息（view.json）  
>> 6.视频评论（comment.json）  
>> 7.视频封面（cover.jpg）  

> 音频  
>> 1.音频本体（audio.m4a）  
>> 2.音频信息（info.json）  
>> 3.音频tag（tag.json）  
>> 4.音频封面（cover.jpg）  
>> 5.音频评论（comment.json）
